## Autoras

## Descripción
Este proyecto implementa un árbol estático con análisis y visualización de resultados.

## Estructura del Proyecto

### 📄 `static_tree.cpp`
Contiene la **implementación** completa del algoritmo de árbol estático en C++. Aquí se encuentra toda la lógica y funcionalidad del programa.

### 📊 `resultados y explicación.html`
Archivo HTML con los **resultados obtenidos** y una **explicación detallada** del funcionamiento del algoritmo y los análisis realizados.

### 📈 `results.csv`
Archivo CSV con los datos de resultados en formato tabular para análisis y procesamiento.

## Cómo usar
1. Consulta la implementación del algoritmo en `static_tree.cpp`
2. Abre `resultados y explicación.html` en el navegador para ver los resultados y la explicación completa
3. Los datos en formato CSV están disponibles en `results.csv`
